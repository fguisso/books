Este projeto hospeda múltiplos livros. O Hugo gera as páginas HTML e o tema [Hextra](https://github.com/imfing/hextra) fornece o layout.

Além da versão web, uma _GitHub Action_ converte os capítulos Markdown em arquivos PDF e EPUB com o Pandoc e LaTeX. Os binários finais são publicados em GitHub Pages junto com o site estático.
