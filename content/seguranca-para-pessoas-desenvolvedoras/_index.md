O livro "Segurança para Pessoas Desenvolvedoras" está em construção e é um projeto de conteúdo livre, distribuído sob a licença Creative Commons BY-SA. O objetivo é oferecer um material acessível e prático para ajudar devs a aplicar princípios de segurança no dia a dia.

Este livro sempre será de acesso livre. Futuramente haverá uma opção paga simbólica e possivelmente uma versão impressa para quem quiser ter em físico.

O projeto já está aberto para feedbacks e colaborações! Para contribuir ou sugerir melhorias, acesse as discussões no GitHub.
